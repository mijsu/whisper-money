# Whisper Money

Privacy-first personal finance application to track, categorize, and understand your spending.

## Tech Stack

- **Backend:** Laravel 13, PHP 8.4
- **Frontend:** React 19, Inertia.js v2, TypeScript
- **Styling:** Tailwind CSS v4
- **Database:** MySQL
- **Cache/Queue:** Redis

## Running with Docker

```bash
docker compose --profile full up -d --build
```

The application will be available at `http://localhost:8000`.

## Features

- Privacy-first — your data stays yours
- Bank account management
- Transaction categorization (manual + AI-powered)
- Automation rules for auto-categorization
- Financial insights and spending patterns
- Budget tracking
- Cashflow analysis
- Multi-currency support
- PWA support (installable on mobile/desktop)
